# 🔮 TokenForge AI - LLM Token Optimizer

TokenForge AI is a specialized developer tool for reducing LLM token usage, improving prompt efficiency, and optimizing structured data (JSON) before sending it to LLM APIs (OpenAI, Anthropic, etc.).

## ✨ Core Features
- **Token Analyzer**: Real-time token counting with `tiktoken` for Gpt-4/Gpt-3.5 models.
- **Token Heatmap**: Visualizes token density to highlight 'expensive' sections of your prompt.
- **TOON Converter**: Eliminates key redundancy in JSON arrays.
- **Prompt Optimizer (NLP)**: Semantically reduces prompt length while maintaining original meaning.
- **Accuracy Validator**: Similarity comparison (0-100%) between original and optimized output.
- **Premium UI**: Modern dark-mode React dashboard with glassmorphism and animations.
- **Privacy First**: Local Flask backend processing.

## 🚀 Quick Start
1. **Backend**: `cd backend && .\venv\Scripts\activate && python app.py`
2. **Frontend**: `cd frontend && npm install && npm run dev`
3. **Chrome Extension**: Load the `chrome_extension` folder as an unpacked extension in Chrome.
